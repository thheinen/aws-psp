## Usage

```ruby
name 'my-cookbook'
maintainer 'John Doe'
maintainer_email 'jdoe@example.com'
issues_url 'https://github.com/jdoe/my-cookbook'
source_url 'https://github.com/jdoe/my-cookbook'
license 'Apache-2.0'
description 'My AWS cookbook'
version '0.1.0'
chef_version '>= 18.0.0'

depends 'aws-psp', '>= 0.1.0'
```
